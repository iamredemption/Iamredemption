Enter your API tokens for each platform below. These are saved in your browser only — never sent anywhere except directly to the platform APIs.

🤖 AI (Anthropic) Not Set

Get your free key at console.anthropic.com → API Keys → Create Key. Required for AI post generation.

📘 Facebook Not Connected

Get your token at developers.facebook.com → My Apps → Create App → Add "Pages" product → Generate Page Access Token.

📷 Instagram Not Connected

Uses the same Meta token as Facebook. Find your Instagram Account ID in Graph API Explorer → query me/instagram_accounts

💼 LinkedIn Not Connected

Get token at developer.linkedin.com → Create App → Request "Share on LinkedIn" + "Organization Social Action" permissions → Generate token.

𝕏 X / Twitter Not Connected

Requires a paid X Developer account ($100/mo Basic tier for posting). Get token at developer.twitter.com.

▶️ YouTube Not Connected

Get token via Google Cloud Console → Enable YouTube Data API v3 → Create OAuth credentials → Authorize. YouTube posts community updates (requires 500+ subscribers).
